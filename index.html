Configuracion de la base de datos

1. Actualizar paquetes

sudo apt update && sudo apt upgrade -y

`sudo apt update`: Actualiza la lista de paquetes disponibles y sus versiones `sudo apt upgrade -y`: Instala las versiones mas recientes de todos los paquetes instalados en el sistema

Instalar PostgreSQL

sudo apt-get postgresql postgresql-contrib libpostgresql-jdbc-java -y

`sudo apt-get postgresql postgresql-contrib libpostgresql-jdbc-java -y`: Instala PostgreSQL, herramientas adicionales y el controlador JDBC y PostgreSQL
